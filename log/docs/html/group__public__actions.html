Метод публикации поста

AUTH: author

Метод публикует пост от автора в пространстве памяти хоста. Каждый пользователь может использовать своё пространство в памяти для хранения постов.

Parameters
[in]ownerПространство в памяти для хранения поста (хост)
[in]authorАвтор поста
[in]permlinkСтроковая прямая ссылка
[in]parent_authorАвтор родительского поста (не обязательно)
[in]parent_permlinkСтроковая прямая ссылка родительского поста (не обязательно)
[in]titleЗаголовок поста
[in]bodyТело поста
[in]metaМета-данные поста
[in]comments_is_enabledИндикатор режима комментирования (вкл/вкл)
[in]priorityПриоритет поста
[in]is_encryptedУказатель шифрованного поста
[in]public_keyПубличный ключ для дешифровки поста
